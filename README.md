# Windows 10 Online Emulator

Windows 10 Online Emulator is a website where you take a experience of Windows 10 in your broswer.

- [x]  Windows Start Menu
- [x]  Windows Search Menu
- [x]  Microsoft Edge
- [x]  Microsoft Store
- [x]  Minecraft
- [x]  Adobe Photoshop
- [x]  Visual Studio Code
- [ ]  This Pc
- [ ]  Recycle Bin
- [ ]  Microsoft Word
- [ ]  Microsoft Excel
